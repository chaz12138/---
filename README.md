# 天猫用户复购预测 | 天池大赛练习项目

基于阿里巴巴天池大赛数据，练习数据清洗、特征工程和机器学习基础流程。

## 项目内容

**数据处理**
- 解析260万+用户行为日志（字符串分割提取购买、点击、时间信息）
- 构建基础特征：activity_count（行为数）、purchase_rate（购买率）、span（时间跨度）
- 用户画像聚合：计算用户全局购买率

**建模尝试**
- 使用逻辑回归模型
- 处理类别不平衡（复购率仅6%）
- 最终AUC：0.6175（优于随机猜测0.5，但仍有提升空间）

**关键发现**
- purchase_count（购买次数）权重最高，是复购的最强预测信号
- 模型过于保守，预测复购率远低于实际，需优化阈值或尝试其他算法
